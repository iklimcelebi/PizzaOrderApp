# 🍕 Pizza Sipariş Uygulaması

Bu proje, **Java Swing** kullanılarak geliştirilmiş bir pizza sipariş uygulamasıdır.  
Kullanıcılar pizzalarını seçebilir, boyut ve hamur türü belirleyebilir, ekstra malzemeler ekleyebilir ve siparişlerini sepetlerine ekleyebilirler.

---

## 🛠️ Özellikler

- Farklı pizza türlerini listeleme
- Pizza boyutu ve hamur tipi seçimi
- Ekstra malzeme ekleme
- Sepet yönetimi
- Sipariş özeti görüntüleme
- Fiyat hesaplama

---

## 💻 Gereksinimler

- Java 17 veya üstü
- Windows, macOS veya Linux işletim sistemi
- Git (opsiyonel, proje yönetimi için)
